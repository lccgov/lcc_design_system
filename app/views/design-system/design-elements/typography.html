For Leeds.gov.uk domains, always use the Open Sans font in Regular for body text, and Montserrat External link in Bold for headings.

Headings

  • use sentence case for headings
  • use headings consistently to create a clear hierarchy
  • use semantic headings and structure. Read article on heading structureExternal link

Lead paragraph

There should only be one lead paragraph per page.

This is a lead paragraph. Pretium molestie ligula in, volutpat mattis libero. Aenean dapibus augue id dolor egestas, in tincidunt dolor volutpat.

This is a body copy paragraph. Felis at varius tincidunt, turpis dui finibus risus, ac varius libero eros ac dolor.

See code 
                
                    <p class="lead">This is a lead paragraph. Pretium molestie ligula in, volutpat mattis libero. Aenean dapibus augue id dolor egestas, in tincidunt dolor volutpat.</p>
                    
                    <p>This is a body copy paragraph. Felis at varius tincidunt, turpis dui finibus risus, ac varius libero eros ac dolor.</p>

Body copy

Avoid using bold and italics.

This is a body copy paragraph. Pretium molestie ligula in, volutpat mattis libero. Aenean dapibus augue id dolor egestas, in tincidunt dolor volutpat.

A supporting text paragraph. Felis at varius tincidunt, turpis dui finibus risus, ac varius libero eros ac dolor.

Links

Links without surrounding text should not have a full stop at the end.

A link without surrounding text

A body copy link. Pretium molestie ligula in, volutpat mattis libero. Aenean dapibus augue id dolor egestas, in tincidunt dolor volutpat.

A body copy external link External link. Fusce dapibus, tellus ac cursus commodo.

A body copy PDF document (PDF, 1MB) External link. Fusce dapibus, tellus ac cursus commodo.

Use bold text with an exclamation icon if there are legal consequences - for example, a fine or prison sentence.

Warning You can be fined up to £5,000 if you don’t register.

Hidden text (progressive disclosure)

Use this to make your page easier to scan, only showing contextual information when required.

Click on "Help with nationality" to see how this works.

Help with nationality

If you’re not sure about your nationality, try to find out from an official document like a passport or national ID card.

We need to know your nationality so we can work out which elections you’re entitled to vote in. If you can’t provide your nationality, you’ll have to send copies of identity documents through the post.
