Promo sizes

The default width of a promo is 100%.

To set the width of a promo, add a class to mega-promo__item. For example, adding the class half will set the width to 50%.

List of classes

Promo width classes
Class Width
featured 100%
two-thirds 66.66%
half 50%
third 33.33%
quarter 25%
